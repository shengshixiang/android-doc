# 配置

* vim ~/.gitconfig

```
[core]
    editor = vim 
[commit]
    template = /home/xielx/git_template/git_template
[user]
    name = xielianxiong
    email = xielianxiong@paxsz.com
[alias]
    ck  = checkout
    st = status
    br = branch
    cp = cherry-pick
    lns = log --name-status
    lpf = log --pretty=fuller
[pull]
    ff = only
```
