README¶

公司要求一套固件,兼容nfc 与 没有nfc的项目.

所以不能直接从编译关闭

要开机检测动态属性,关闭

方法¶

  • 动态检测nfc,关闭 nfc hal 服务vendor.nfc_hal_service, 关闭 service vendor.secure_element_hal_service

    • log如下
