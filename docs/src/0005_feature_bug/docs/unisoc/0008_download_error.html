问题原因¶

根据硬件贴片,gpio87 跟gpio 33,都应该要拉高,才能匹配到分离式 DDR4X

但是V02硬件,gpio33 用作了gpio输出,所以需要硬件先改一下

解决¶

硬件把gpio33,跟gpio87引脚配置后,就正常下载
