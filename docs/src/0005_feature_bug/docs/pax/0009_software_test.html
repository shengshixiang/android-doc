pax 软件提测注意事项

版本发布,发布版本的时候,要完成一些自测试报告,并且了解测试还有哪些阻塞的.

包括:

  • 硬件测试

  • 软件系统测试

  • 无线测试

pax软件测试,硬件测试,完成注意事项

  • 了解哪些模块没有测试

  • 了解哪些用例阻塞测试

  • 重点问对应测试人员,以下模块还有哪些bug,是不是都测试完了.有没有阻塞,要尽早报bug

    • Camera,涉及到是否换模组,项目开始阶段,要留golden,决定是否开otp

    • BMS,bms测试时间长,测试模式,要7天学习,还有一些电池参数配置好

    • 音频, 对应 免提,通话,手持,播放音乐等模式,是否有问题,这个可以很早测试, mic位置对不对,双mic的项目

    • LCD, 亮度最大值,最小值是否符合标准

    • 扫码, 扫码包括静态扫码,成功率,运动容差都测试

    • 功耗,打开wifi,插入sim卡,测试平均电流,还有不插卡底电流
