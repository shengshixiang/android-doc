README¶

PAX tms ,nextgen 后台配置,配置model,下载链接等

配置型号¶

  • 选择导入

  • 选择下载模板

  • 填写型号

  • 上传文件

  • 提交不要覆盖现有记录

  • 创建成功

创建应用下载(可选)¶

  • 添加应用

  • 选择型号

  • 选择apk

  • 等待上传完成,点击提交

  • 成功

上传ota固件¶

  • 添加固件

  • 选择型号,ota类型

  • 选择固件,上传

  • 填入版本号,提交

  • 成功,固件描述名称不能太长,否则会提示错误

终端id管理¶

  • 新建下载id

  • 填入下载id,目标型号,点击下一步

  • 选择应用(可选)

  • 选择需要的应用,提交,点击下一步

  • 选择固件(可选)

  • 提交

  • 成功

下载¶

  • 安装应用PaxTmsDemo

  • 填入port,ip,terminal id,点击下载

    terminal id 就是nextgen 创建的terminal id
